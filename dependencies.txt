Fonts:

    JetBrains Mono Fonts (ttf-jetbrains-mono)
    [for i3 setup also download any nerd font]
    -- or --
    Iosevka (ttf-iosevka-nerd)

    Victor Mono (font-victor-mono {aur} ||
                 nerd-fonts-victor-mono {aur})
    [for dwm setup; better to be downloaded
     from official website as I currently
     use it with Font Awesome (ttf-font-awesome)]
    
    -> noto-fonts (noto-fonts noto-fonts-cjk noto-fonts-emoji)
    [I suggest installing all noto-fonts
     packages if you want all characters and
     emoji to be shown]

WMs:

    -> i3 (i3-gaps i3status i3lock rofi)
    -- or --
    -> dwm (dwm dmenu)
    [both dwm and dmenu are better
     to be downloaded from suckless.org]

Terminals:

    kitty
    [terminal emulator I'm using now
     instead of alacritty as it seems
     to work improperly with X11]

Console Applications:

    nvim (neovim)
    [modern vim replacement]

    exa
    [modern ls replacement with icons
     note: icons are shown properly
     only with nerd fonts so you either
     need to reject using icons or edit
     them yourself in exa source code;
     I consider editing exa myself later]

    ranger
    [simple terminal file manager with vim keybindings]

    zsh
    [better shell; use with ohmyzsh framework]

Utils:

    pcmanfm
    -- or --
    thunar
    [GUI file manager, if you need one]

    scrot
    [simple screenshot utility]

    sxiv
    [simple X image viewer]

    -> zathura (zathura zathura-pdf-mupdf zathura-djvu)
    [simple pdf and djvu reader with vim keybindings]

    picom
    [compton fork; screen compositor]

    xdg-user-dirs
    [Desktop, Downloads, etc folders support]

Theming:

    lxappearance
    [GUI-based utility for setting up gtk themes and fonts]

    arc-gtk-theme
    [available in Arch repos]

    arc-icon-theme
    [also available in Arch repos]

    Twilight-cursors (ArcMidnight-cursors, LyraQ-cursors)
    [download manually from gnome-look for example
     and put into ~/.icons folder]
