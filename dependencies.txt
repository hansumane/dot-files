Console Applications:

    nvim (neovim)
    [modern vim replacement]

    [better shell; use with ohmyzsh framework]

Theming:

    qt5ct
    [qui-based utility to set up qt themes in windows managers
     you should append 'export export QT_QPA_PLATFORMTHEME="qt5ct"'
     to /etc/profile]
    [you only need it if you don't use KDE or LXQT]

    lxappearance
    [GUI-based utility for setting up gtk themes and fonts]
    [you only need it if you don't have theme configurer
     in your desktop environment]

    arc-gtk-theme
    papirus-icon-theme

    [cursors are included in ./themes folder]

Other:

    font-patcher (https://github.com/ryanoasis/nerd-fonts.git)
    [utility to patch any font with nerd fonts icons]
