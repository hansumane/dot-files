Fonts:

    JetBrains Mono Fonts (ttf-jetbrains-mono)
    
    any nerd font installed for icons to appear
    [I prefer to use either Hack, Meslo, Iosevka or
     you can even download JetBrains Nerd Font
     but I prefer JetBrains Mono being installed
     with package manager and not manually]
    
    noto-fonts*
    [I suggest installing all noto-fonts
     packages if you want all characters and
     emoji to be shown]
